# FATSA/FASTQ Parser in R

## Overview
This is a beginner bioinformatics project written in R to parse and analyze FASTA and FASTQ files, to gain practice working with biological sequence data. 

## Project Status

This project is currently in progress

Completed: 
- FASTA file parser
- FASTQ file parser

In Progress: 
- GC content calculation 

Planned: 
- Sequence length summary
